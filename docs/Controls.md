# Rotary Encoder
    The rotary encoder is used for real-time volume control.
       Clockwise - Increases Volume
       Anticlockwise - Decrease Volume
       Push Button is used to select song (Press once to scroll through songs and press again to select song)

# Push Buttons
    The system utilizes three tactile push buttons for playback control. 
      Play/Pause Button (GP2): Toggles the current audio playback state between playing and paused. 
      Next Track Button (GP3): Stops the current track and loads the next available file in the SD card. 
      Previous Track Button (GP4): Stops the current track and restarts the previous file in the sequence.
